## String 메서드
- subString(1) : 앞에서부터 매개변수만큼 문자열을 자른 뒤 반환.
    - subString(str.length()-1) : 문자열의 끝 문자열 앞까지 자르고 반환.
    - charAt(str.length()-1) : 문자열의 끝 문자 반환.
- charAt(1) : 인덱스 1번 문자 반환.
- A.compareTo(B) : A와 B를 유니코드로 비교. 같으면 0, A < B 면 음수, A > B 면 양수 반환.
